# ArknightsAppData

明日方舟CN-简中服部分文件数据。

与官方数据同步。

# 目前支持的
- 手机下载补丁包后的Android文件数据库，目前暂无计划添加apk包文件库
- 当前游戏内大部分文本弹窗提示
- 游戏内绝大部分BUFF模板数据（同理，并非apk包内数据）

后续可能会添加更多东西，看情况定
